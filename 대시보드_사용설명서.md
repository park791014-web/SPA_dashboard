# 부산동성고등학교 성적 분석 대시보드 사용설명서

본 프로그램은 나이스(NEIS) 학기말 종합일람표 및 전입생용 학교생활기록부 교과학습발달사항 엑셀 데이터를 종합 분석하여, 학생별 3개년 성적 변화 추이와 과목군별 성적 분포를 다각도로 시각화해 주는 교사용 대시보드입니다.

---

## 🌟 핵심 기능 요약

1. **다중 엑셀 파일 동시 분석 및 자동 분류**:
   - 나이스 종합일람표 포맷과 생활기록부 교과학습발달사항 포맷을 이름/내용 기반으로 웹 브라우저가 자동 구분하여 파싱합니다.
2. **전입생 임시 학번 자동 매칭 및 병합**:
   - 전입생의 임시 학번(예: `19912`)과 정규 배정 학번(예: `10217`, `20712`)을 크로스 체크하여 동명이인 분리 오류 없이 단 하나의 학생 프로필로 성적을 완벽 병합합니다.
3. **정밀 누락/제외 감지 경고 시스템**:
   - **`누락` (빨간 뱃지)**: 최상위 재학 학년 대비 이전 학년의 성적이나 특정 학기가 비어 있는 활성 전입생을 표시합니다. 누락 학년은 빨간 글씨로 뱃지가 렌더링됩니다.
   - **`제외` (회색 뱃지)**: 과거 성적은 있으나 현재 학년 성적이 없어 전출/자퇴로 판단되는 학생을 자동 분류합니다.
4. **3차원 분석 그래프 시각화**:
   - **과목군별 평균 등급** (방사형 레이더 차트)
   - **과목군별 등급 분석** (과목군 및 등급 분포 누적 막대 차트)
   - **학기별 등급 변화 추이** (학년/학기별 변화 선 차트 - 종합 평균 검정색 표시)
5. **완벽한 오프라인 작동**:
   - 모든 외부 라이브러리(SheetJS, Chart.js, Lucide Icons)를 프로그램 폴더에 탑재하여, **인터넷 연결이 차단된 학교 교내 인트라넷 PC에서도 100% 가동**됩니다.
6. **보안성 극대화**:
   - 업로드된 학생들의 이름, 학번, 성적 등 모든 개인정보는 서버로 가지 않으며 **선생님의 PC 웹 브라우저 메모리상에서만 연산된 후, 창을 닫으면 흔적 없이 완전히 소멸**합니다.

## 📂 지원 파일 양식 및 준비법

대시보드에 업로드할 수 있는 파일 규격은 크게 세 가지입니다.

### A. 학기말 성적 종합일람표 (기본 일람표)
- 나이스(NEIS)에서 내려받는 표준 성적 종합일람표 엑셀 파일입니다.
- 파일명 예시: `2025_1학기_학기말_101.xlsx` (파일명에 `학년`, `반`, `학기` 정보가 자연스럽게 포함되는 것을 권장합니다.)

### B. 전입학용 생활기록부 교과학습발달사항 파일
- 전입생이 이전 학교에서 취득한 학업 기록을 반영하기 위한 파일입니다.
- **포맷 구조**: 첫 행에 `성 명`, `학 년`, `학 기`, `과 목`, `학 점(단위수)`, `석차등급` 컬럼이 배치되며, 병합 셀 형태로 학생 이름이 지정됩니다.
- **다중 학생 기입 가능**: 동일 파일 아래쪽에 다른 전입 학생 정보를 이어서 작성(누적)해도 자동으로 각각 식별하여 반영합니다. 파일명은 자유롭게 지으셔도 무방합니다.

### C. 직접 붙여넣기 (텍스트 복사)
- 엑셀 표 영역을 블록 지정하여 복사(`Ctrl+C`)한 후, 대시보드의 "직접 붙여넣기" 탭에 `Ctrl+V`하여 즉석 분석이 가능합니다.

## 🚦 화면 표시 가이드

### 1. 학생 목록 (좌측 사이드바)
- **`제외` (회색 뱃지)**: 전출/자퇴 추정 학생. 이전 기록은 있으나 최신 학년 성적이 수합되지 않은 학생입니다.
- **`누락` (빨간 뱃지)**: 이전 학년 성적 또는 특정 학기 성적이 누락된 학생입니다. 
  - 마우스 커서를 올리면 누락된 학년/학기 정보(예: `1학년 전체` 또는 `2학년 2학기`)가 툴팁으로 표시됩니다.
- **빨간색 학년 글씨**: 학년 정보 목록 중 글씨가 빨간색으로 표시된 학년은 해당 학년에 일부 성적이 누락되었음을 의미합니다.

### 2. 학생 상세 프로필 (우측 메인 카드)
- 프로필 최상단의 학년별 학번 태그 영역에서 누락 상태를 색상으로 안내합니다.
  - **빨간색 점선 테두리 (`미입력 (자료 누락) ⚠️` 또는 `(2학기 누락) ⚠️`)**: 전입학 과거 서류가 입력되지 않았거나 특정 학기가 비어 있는 경고 상태입니다.
  - **회색 태그 (`미기록`)**: 미래 학년이거나 정상적인 전출 처리 상태입니다.

## 🛠️ 문제 해결 (Q&A)

**Q. 파일을 선택했는데 대시보드에 아무런 변화가 없습니다.**
> **A**: 
> 1. 웹 브라우저가 예전 캐시 파일을 사용 중일 수 있으니 대시보드 창에서 **`Ctrl + F5` (강제 새로고침)**를 누른 후 재시도해 보세요.
> 2. 업로드한 엑셀 파일 중 시스템 임시 파일(`~$`로 시작하는 파일)이나 이미지 등 엑셀 포맷이 아닌 파일이 섞여 있는 경우 자동 스킵됩니다. 정식 `.xlsx` 파일만 모아둔 폴더를 선택했는지 확인해 주세요.

**Q. 전입생의 1학년 성적이 학번에 맞게 병합되지 않고 2학년 자료와 따로 분리되어 뜹니다.**
> **A**: 전입생용 엑셀 파일에 적힌 이름과 학기말 일람표에 적힌 **이름의 띄어쓰기나 철자가 다르면** 동명이인으로 판단하여 카드가 분리될 수 있습니다. 엑셀 원본 파일에서 학생 이름(예: `동성고` vs `동 성고`)이 완전하게 일치하는지 확인해 주세요.

**Q. 엑셀로 내보내기 버튼은 언제 나타나나요?**
> **A**: 성적 데이터를 1개 파일 이상 정상 로드하면, 화면 우측 상단 헤더 영역에 초록색 **`엑셀로 저장`** 버튼과 빨간색 **`대시보드 초기화`** 버튼이 동적으로 나타납니다.
